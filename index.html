Sobre o HTML

Criada pelo britânico Tim Berners-Lee, o acrônimo HTML significa HiperText Markup Language, traduzindo ao português: Linguagem de Marcação de Hipertexto.
O HTML é o componente básico da web, ele permite inserir o conteúdo e estabelecer a estrutura básica de um website. Portanto, ele serve para dar significado e organizar
as informações de uma página na web. Sem isso, o navegador não saberia exibir textos como elementos ou carregar imagens e outros conteúdos.

Os hipertextos são conjuntos de elementos conectados. Esses podem ser palavras, imagens, vídeos, documento, etc. Quando conectados, formam uma rede de informações
que permite a comunicação de dados, organizando conhecimentos e guardando informações.

Tags e para que servem?

As tags servem para definir a estrutura da página, por exemplo: tamanho de fonte do texto, espaçamentos, videos e imagens. Todo comando ou texto deve ficar entre as
tags para que a ação desejada aconteça.

Tags - Elementos HTML

  • <!DOCTYPE html> : Serve para identificar o tipo de documento (nesse caso html);

  • <html></html> : Este é a base para a página HTML, tudo deve ficar dentro dessa tag.

  • <tytle></title> : É o titulo da página, que vai aparecer na barra do navegador;

  • <head></head> : Ele contem meta informações sobre a página;

  • <body></body> : Tudo o que estiver entre ele vai aparecer para o usuário;

  • <p></p> : Insere textos;

  • <div></div> : Usado para dividir os atributos;

  • <h1></h1> : Inserir titulo do texto, a numeração vai de 1 a 6, cada definindo diferentes tamanhos para o titulo;

  • <ol></ol> : Cria uma lista ordenada (númerada);

  • <u1></u1> : Cria uma lista desordenada;

  • <li></li> : Usada para incluir os itens de uma lista;

  • <a></a> : Serve para incluir um link que leva a um site diferente;

  • <blockquote></blockquote> : Usado para fazer uma sitação;

  • <strong></strong> : Deixa o texto em negrito;

  • <i></i> : Deixa o texto em italico;

  • <mark></mark> : Grifa o texto onde for colocada;

  • <!----> : Serve para fazer anotações dentro do html, mas não aparecem no navegador;

  • <u></u> : Sublinha uma palavra, frase ou texto;

  • <hr> : Cria um traçado logo abaixo do texto.

Tags - Atributos HTML

(Voltar)

Estes servem para adicionar novos atribulos aos elementos de HTML

  • href="" : Usado para incluir uma URL;

  • img="" : Inclui uma imagem a página;

  • title="" : Acrescenta um texto que aparece ao levar o cursor até o link;

  • id="" : Identifica um link usado para se deslocar em uma mesma página.
